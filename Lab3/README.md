# Lab 3: JavaScript Calculator

## Description
A fully functional calculator where every element and every style is created entirely via JavaScript — no HTML markup and no CSS files.

## Tasks Completed
- All DOM elements created with `document.createElement()`
- All styles applied via `.style` property
- Responsive layout (mobile / tablet / desktop) via `window.innerWidth` + `resize` listener
- Operations: `+`, `−`, `×`, `÷`
- Extra: `+/−` sign toggle, `%` percentage, `AC` clear, keyboard support, backspace

## How to Run
Open `index.html` in any modern browser. The script tag is the only thing in `<body>`.

## Technologies Used
- HTML5 (minimal shell)
- Vanilla JavaScript (ES6+) — all layout and styling

## Features
- iOS-inspired dark theme built entirely in JS
- Keyboard input (digits, operators, Enter, Backspace, Escape)
- Division-by-zero guard (shows "Error")
- Responsive: adjusts button/font size for mobile, tablet, desktop
