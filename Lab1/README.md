# Lab 1: DOM Events & Event Handlers

## Description
Building dynamic web pages using events and the Document Object Model (DOM).

## Tasks Completed
1. **Image Click Handler** – 3 images bound with `elem.onclick`; clicking logs the image's `width` attribute to the console.
2. **Link Hover Handler** – 3 links use `addEventListener`/`removeEventListener`; on `mouseenter` the link's `title` is set to its `href`.
3. **Input Value Display** – 3 inputs; clicking any input displays its `value` in `#demo`.
4. **First-Click vs Repeat-Click** – first click logs to console, all subsequent clicks show an `alert()`. Implemented by swapping event listeners.
5. **Number Square Calculator** – 3 paragraphs with number words; clicking replaces the word with the square of that number (e.g. "five" → 25).

## How to Run
Open `index.html` in any modern browser. Open DevTools console to see log output.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
