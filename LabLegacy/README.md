# Lab Legacy (2023): JavaScript Fundamentals

## Description
Conditional constructs and loops in JavaScript.

## Tasks Completed
1. **Prime Numbers** – outputs all primes from 0–100 using a `while` loop.
2. **Even/Odd Classifier** – `do...while` loop classifies 0–10 as zero / odd / even (Ukrainian labels).
3. **Division Loop** – starts at 10000, divides by 2 until result < 50; logs final value and iteration count.
4. **Season Detector** – enter a month number (1–12) to get the season and month name.
5. **Temperature Converter** – converts Celsius to Fahrenheit using `Tf = (9/5)*Tc + 32`.
6. **Day of Week** – enter 1–7 to see the corresponding weekday name.

## How to Run
Open `index.html` in any modern browser. All tasks have interactive buttons and input fields.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
