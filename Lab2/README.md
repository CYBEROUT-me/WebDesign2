# Lab 2: Color Toggle with Event Listeners

## Description
Demonstrates the `removeEventListener` pattern by alternating between two click handlers.

## Tasks Completed
1. **Color Toggle Divs** – 3 divs start green. First click paints red (swaps to `paintGreen` handler). Second click paints green (swaps back to `paintRed` handler). Repeats indefinitely.

## How to Run
Open `index.html` in any modern browser and click the divs.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
